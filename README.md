# Analyzing-Website-Performance-for-The-Grammys
Utilized advanced Excel analytics, PivotTables, conditional logic (IF, SUMIFS, AVERAGEIFS), and XLOOKUP, to calculate user engagement KPIs and mobile visitor rates, comparing performance of Grammy.com and RecordingAcademy.com to assess the business impact of their 2022 website split.
# 📊 Grammy vs. Recording Academy Website Split Analysis

## 🎯 Project Overview
An Excel-based case study investigating user behavior, engagement metrics, and traffic shifts following the 2022 website separation between Grammy.com and RecordingAcademy.com. 

## 🛠️ Technical Skills Demonstrated
* **Data Cleaning & Management:** Organized raw web traffic logs, removing null errors and formatting timelines for clean reporting.
* **PivotTable Modeling:** Built automated aggregation models using conditional formulas (`IF`, `SUMIFS`, `AVERAGEIFS`) and `XLOOKUP`.
* **Business Intelligence:** Tracked and visualized a sharp drop in user attention span on Grammy.com (average time on site fell from 128.50s to 82.99s).
